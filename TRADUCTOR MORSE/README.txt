INSTRUCCIONES 
1.-Descargar el archivo TraductorMorse.zip 
2.-Descomprimir y abrir el proyecto en NetBeans
3.-Ejecutar el proyecto (clase TraductorMorse.java)
4.-Seleccionar una de las opciones mostradas en el menu de la consola
5.-Escribir el texto a traducir( morse o texto normal ) y presionar la tecla enter
6.-El programa mostrara la traducción del texto ingresado
7.-Seleccionar opcion 4 para termiar la ejecución
